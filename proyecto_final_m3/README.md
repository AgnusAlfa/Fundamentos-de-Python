# Proyecto Final Módulo 3 - Ecommerce CLI

# Descripción del Programa
Este proyecto es una aplicación de consola (CLI) desarrollada en Python que simula el funcionamiento de un Ecommerce. 
El sistema permite a los usuarios interactuar con un menú principal continuo para realizar las siguientes acciones:
* Ver un catálogo de productos estructurado.
* Buscar artículos específicos filtrando por nombre o categoría.
* Agregar productos a un carrito de compras interactivo con validación de cantidades.
* Visualizar el detalle del carrito, incluyendo subtotales por producto y el total final a pagar.
* Vaciar el carrito de compras.

El código fue construido aplicando los fundamentos de programación en Python, destacando el uso de diccionarios (para el catálogo), listas (para el carrito), bucles `while` y `for`, manejo de excepciones con `try/except` para prevenir caídas del sistema, y una arquitectura basada en funciones modulares.

## Cómo ejecutarlo
1. Asegúrate de tener Python instalado en tu sistema.
2. Abre la carpeta del proyecto en Visual Studio Code (o tu editor de preferencia).
3. Abre el archivo principal llamado `ecommerce_m3.py`.
4. Haz clic derecho sobre cualquier parte del código y selecciona la opción **"Run Python File in Terminal"** (o presiona el botón de "Play" en la esquina superior derecha).
5. La interfaz se desplegará en la terminal inferior. Interactúa con el menú escribiendo el número de la opción deseada y presionando `Enter`. Para finalizar la ejecución, selecciona la opción `0`.
