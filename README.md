# Fundamentos-de-Python

RESPUESTAS ACTIVIDAD 5 - Explorando Estructuras de Datos

1. ¿Cuál estructura te pareció más flexible?
La lista (list) me pareció la más flexible. Esto se debe a que es una estructura "mutable", lo que significa que nos permite agregar nuevos elementos (usando append), eliminar elementos existentes (usando remove) y modificar sus valores en cualquier momento de la ejecución. Además, mantiene un orden claro a través de sus índices.

2. ¿Para qué tipo de problema usarías un diccionario en lugar de una lista?
Usaría un diccionario para problemas donde necesito organizar y buscar información utilizando etiquetas descriptivas (claves) en lugar de posiciones numéricas (índices). Por ejemplo, si quiero guardar los datos de un producto (como marca, modelo y año), un diccionario me permite acceder directamente al valor escribiendo diccionario["marca"], lo cual es mucho más intuitivo y rápido que intentar recordar en qué número de posición de una lista guardé esa información.
